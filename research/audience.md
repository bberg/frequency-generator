# Target Audience Analysis: Frequency Generator

## Primary Audiences

---

## 1. Audio Engineers (Studio & Live)

### Demographics
- Professional or semi-professional
- Age 25-55
- Technical background
- Regular need for calibration tools

### Use Cases
- **Daily:** Reference tone generation, quick speaker checks
- **Weekly:** System calibration, frequency response testing
- **As needed:** Troubleshooting audio issues, phase problems

### Pain Points
- Need quick, reliable tools without software installation
- Existing tools often require accounts or subscriptions
- Need precise, calibrated signals
- Want to test equipment before/during sessions

### Features They Need
1. 1kHz and 440Hz reference tones
2. Pink noise for speaker tuning
3. Frequency sweeps for response testing
4. Phase/polarity testing
5. Stereo L/R verification
6. Precise frequency control
7. Multiple waveform types

### How to Reach Them
- Audio engineering forums (Gearslutz, Reddit r/audioengineering)
- DAW communities
- Professional audio publications
- YouTube tutorials

---

## 2. Audiophiles & Hi-Fi Enthusiasts

### Demographics
- Hobbyist to serious enthusiast
- Age 30-65
- Invested in high-quality audio equipment
- Active in online communities

### Use Cases
- Testing new speakers/headphones
- Verifying frequency response claims
- Room acoustics exploration
- Comparing equipment

### Pain Points
- Want to verify manufacturer specs
- Need to test bass extension
- Want to identify equipment limitations
- Room mode problems

### Features They Need
1. Wide frequency range (especially low end)
2. Frequency sweeps
3. Bass extension tests
4. High-frequency tests (age-related hearing)
5. Stereo imaging tests
6. Quality visualizations

### How to Reach Them
- Head-Fi forums
- r/audiophile
- Audio review sites
- Hi-Fi YouTube channels

---

## 3. Musicians & Music Students

### Demographics
- Age 15-50
- Instrument players, vocalists, producers
- Various skill levels
- Need tuning and pitch references

### Use Cases
- Instrument tuning (A440, alternative pitches)
- Ear training
- Understanding frequency and pitch relationship
- Practice tools

### Pain Points
- Need quick reference tones
- Want to understand frequency/pitch correlation
- Learning audio fundamentals
- Testing monitoring equipment

### Features They Need
1. Musical note presets (A440, etc.)
2. Alternative tuning references (432Hz)
3. Equal temperament note chart
4. Visual frequency-to-note display
5. Educational content about pitch

### How to Reach Them
- Music education platforms
- Instrument-specific forums
- Music subreddits
- YouTube music channels

---

## 4. People Concerned About Hearing Loss

### Demographics
- Age 30-70
- May have noticed hearing changes
- Concerned about age-related decline
- Seeking self-assessment tools

### Use Cases
- Testing highest audible frequency
- Comparing hearing between ears
- Monitoring changes over time
- Pre-audiologist screening

### Pain Points
- Professional hearing tests are expensive
- Want quick, private self-assessment
- Concerned about high-frequency loss
- May have tinnitus

### Features They Need
1. Hearing range test
2. Age-related hearing frequency guide
3. Left vs. right ear testing
4. Safe volume warnings
5. Educational content about hearing health
6. Tinnitus frequency matching

### How to Reach Them
- Health-focused websites
- Hearing aid forums
- Senior-focused communities
- Tinnitus support groups

---

## 5. Professionals Doing Acoustic Measurements

### Demographics
- Acoustic consultants
- Building engineers
- Sound system installers
- AV integrators

### Use Cases
- Room acoustics testing
- Standing wave identification
- System commissioning
- Equipment verification

### Pain Points
- Need portable, reliable tools
- Expensive specialized equipment
- Need quick verification signals
- Field work requirements

### Features They Need
1. Precise calibration tones
2. Sweep generators
3. Pink/white noise
4. Standard test frequencies
5. Reference levels
6. Downloadable test files

### How to Reach Them
- AES (Audio Engineering Society)
- Acoustic consulting forums
- AV industry publications
- Trade shows and conferences

---

## 6. Educators & Students

### Demographics
- Physics/music teachers
- Students (high school through university)
- Self-learners
- Hobbyist scientists

### Use Cases
- Demonstrating frequency concepts
- Physics experiments
- Understanding sound waves
- Learning audio fundamentals

### Pain Points
- Need free, accessible tools
- Want educational explanations
- Need visual representations
- Classroom-friendly interface

### Features They Need
1. Clear visualizations
2. Educational content
3. Easy-to-understand interface
4. Frequency-to-pitch explanations
5. Waveform comparisons
6. Physics concepts explained

### How to Reach Them
- Educational resource sites
- Teacher forums
- Student communities
- Science education YouTube

---

## 7. DIY/Maker Community

### Demographics
- Electronics hobbyists
- Speaker builders
- Audio DIY enthusiasts
- Makers and tinkerers

### Use Cases
- Testing DIY audio projects
- Speaker driver testing
- Crossover design verification
- Circuit testing

### Pain Points
- Need free, accessible tools
- Want technical accuracy
- Testing homebrew equipment
- Learning audio fundamentals

### Features They Need
1. Wide frequency range
2. Precise control
3. Technical specifications
4. Educational content
5. Frequency response testing
6. THD awareness

### How to Reach Them
- DIYAudio forums
- Parts Express community
- r/diyaudio
