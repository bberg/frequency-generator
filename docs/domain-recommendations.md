## Recommendation

**Primary:** `testtones.com`
- Short, memorable, easy to spell
- Describes core function (testing with tones)
- Broad appeal: speaker testing, calibration, hearing tests
- Professional sound

**Backup:** `hertzgen.com`
- Very short (8 chars)
- "Hertz" = frequency units - recognizable
- Modern, tech-savvy feel

**Alternative:** `calibrationtone.com`
- More professional/technical positioning
- Good for audio engineer audience
